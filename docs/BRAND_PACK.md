# Brand Pack — Students Mentoring

## Brand Identity
- **Name:** MentorConnect
- **Tagline:** "Grow together, learn forever."
- **Voice:** Friendly, encouraging, professional but not corporate.

## Colour Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `primary` | `#4F46E5` | Indigo — buttons, links, active states |
| `primary-light` | `#818CF8` | Hover states, highlights |
| `primary-dark` | `#3730A3` | Text on light backgrounds |
| `secondary` | `#F59E0B` | Amber — accents, badges, CTAs |
| `neutral-50` | `#F9FAFB` | Page background |
| `neutral-100` | `#F3F4F6` | Card backgrounds |
| `neutral-700` | `#374151` | Body text |
| `neutral-900` | `#111827` | Headings |
| `success` | `#10B981` | Positive feedback |
| `error` | `#EF4444` | Errors, destructive actions |

## Spacing & Layout
- Base unit: **4 px** (`0.25 rem`).
- Content max-width: **1 280 px** (`max-w-7xl`).
- Card border-radius: **0.75 rem** (`rounded-xl`).
- Default padding: `p-4` (mobile), `p-6` (md+).

## Iconography
- Use **Lucide React** icons (consistent with shadcn/ui).
- Icon size: 20 px inline, 24 px standalone.

## Logo
- SVG logo stored at `public/logo.svg` and `public/logo-dark.svg`.
- Minimum clear space: 8 px around all sides.

## Imagery
- Illustrations: use diverse, inclusive student/mentor illustrations.
- Photos: warm tones, natural lighting, educational settings.
- Aspect ratios: hero 16:9, cards 4:3, avatars 1:1 (rounded-full).
